Data Science & Machine Learning
Projects
A collection of technical tasks covering Data Visualization, Time Series Analysis, and
Machine Learning.

Overview
This repository showcases a structured approach to data-driven problem solving, ranging
from exploratory data analysis and visualization to predictive modeling and forecasting.

Project Modules
1. Data Visualization (Iris Dataset)
Focuses on exploratory data analysis (EDA) using the classic Iris dataset to understand
feature distributions and relationships.
Tools: Matplotlib, Pandas
Key Visuals: Histograms, Scatter plots, and Box plots of sepal/petal dimensions.
2. Time Series Analysis (Stock Price Prediction)
Analysis of historical stock data (AABA) from 2006 to 2017 to identify market trends and
seasonal patterns.
Key Tasks: Moving averages, trend visualization, and statistical stationarity checks.
Dataset: stock_data.csv
3. Machine Learning (Loan Default Prediction)
A comparative study of classification and regression models to predict loan eligibility and
default risk.
Models Tested: Linear Regression, Decision Tree, Random Forest.
Top Performer: Random Forest Classifier (Accuracy: 66.6%).
Preprocessing: Label Encoding, Mode Imputation, and Feature Scaling.

Tech Stack
Python Pandas NumPy Scikit-Learn Matplotlib Statsmodels

How to Use
To replicate the analysis, clone the repository and install the necessary dependencies:
pip install pandas numpy scikit-learn matplotlib statsmodels
Run the .ipynb files in any standard Jupyter environment.
